# One-Touch Subway
2020 모바일 프로그래밍 프로젝트

이 어플리케이션은 음성인식을 기반으로 지하철 경로를 조회하는 서비스입니다.

1. 음성인식 방법

  가. ~에서 ~
    ex) 상봉에서 중랑
    
  나. ~역에서 ~
    ex) 상봉역에서 중랑역
    
  다. 사용자 지정 위치가 ~역에 대체될 수 있습니다.
    ex) 집 - 중랑, 학교 - 건대입구
        집에서 학교 = 중랑에서 건대입구
        
2. 사용자 지정 위치 등록 방법

  상단 + 아이콘을 누르고 들어가면, 사용자 지정 위치와 역을 지정할수 있는 창이 띄워집니다.
  사용자는 추가, 삭제, 업데이트를 수행할 수 있습니다.
